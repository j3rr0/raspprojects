In Python kun je wiskundige operatoren gebruiken om sommen te maken:
| operator | beschtrijving |
| ----------- | ----------- |
| + | optellen |
| - | aftrekken|
| * | vermenigvuldinging|
| / | delen|
| ** | machtsverheffing|
